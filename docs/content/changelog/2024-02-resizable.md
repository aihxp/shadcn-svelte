### New Component: Resizable

We've added a new component to the project, [Resizable](/docs/components/resizable), which is built on top of [PaneForge](https://paneforge.com). PaneForge is still in an early stage, so be sure to raise any issues you find with the library on the [PaneForge GitHub](https://github.com/svecosystem/paneforge).

### Updated Icon Imports

We've updated the way we import icons. Instead of importing icons from entire collections, component examples now use deep imports for the icons they need.

### Major Forms Update

Formsnap has been rewritten to be more flexible, easier to use, and less opinionated. All of the `Form` components have been updated to use the new API.
